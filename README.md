# Mujagi77 | Logic Hub 🧠

Dobrodošli u **Logic Hub** – digitalnu destinaciju za logičke i matematičke izazove. Ovaj projekat je kreiran sa ciljem da testira vašu logiku, koncentraciju i strpljenje kroz seriju interaktivnih igara.

## 🚀 O projektu
Logic Hub je skup laganih, brzih i optimizovanih igara koje rade direktno u vašem pretraživaču. Nema instalacije, nema reklama – samo čista logika.

## 🎮 Igre koje možete igrati
*   **Logička Vaga** – Izazov za vašu logiku i procenu.
*   **Lavirint** – Testirajte svoju orijentaciju i brzinu.
*   **Magična Karta** – Zabavan trik koji će vas ostaviti u čudu.
*   **Slagalica** – Klasičan test strpljenja i organizacije.

## ✨ Ključne karakteristike
*   **Minimalistički dizajn:** Fokusiran na korisničko iskustvo (Dark mode / Neon stil).
*   **Brzina:** Optimizovan kod bez nepotrebnih skripti.
*   **Responsivnost:** Igre savršeno rade na telefonima, tabletima i kompjuterima.
*   **Audio povratne informacije:** Zvučni efekti za bolji gejming doživljaj.

## 🛠 Tehnologije
Projekat je razvijen koristeći standardne web tehnologije:
*   HTML5
*   CSS3 (sa fokusom na performanse i animacije)
*   JavaScript (Vanilla - bez eksternih biblioteka)
